Summary: Latitude vs. X

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accompish this analysis, we first pulled data form the OpenWeatherMap API to assemble a dataset on over 500 cities.

After assembling the dataset, we used Matplotlib to plot various aspects of the weather vs. latitude. Factors we looked at included: cloudiness, humidity, temperature, and wind speed. This site provides the source data and visulaizations created as part of teh analysis, as well as explanations and descriptions of any trends and correlations witnessed.
